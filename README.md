# 弥生・古墳時代クイズ（小5・3月）

中学受験サポート用のクイズアプリです。小5・3月の演習から、全20問を出題します。

## 出題内容
- 弥生時代の暮らし（弥生土器・高床倉庫・石包丁・登呂遺跡）
- 邪馬台国と卑弥呼（魏志倭人伝）
- 古墳時代（前方後円墳・大仙古墳・はにわ）
- 大和政権・渡来人・ワカタケル大王（稲荷山古墳の鉄剣）
